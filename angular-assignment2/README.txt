README
-------

This is a small front end application built with AngularJS. In this application the user needs to implement a shopping Checklist. 
The user has a prepopulated "To Buy" Shopping List. Next to each item in "To Buy" list is a Bought button.Once the Bought button is clciked the 
item should be removed from "To Buy" list and added to "Already Bought" List

AngularJS version used is v1.6.4
